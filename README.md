🤖 Telegram Hosting Bot (24/7)

A simple multi-file Telegram bot hosting system with auto-restart and 24/7 support.

🚀 Features

- ✅ Upload & Run Python files
- 🔄 Auto Restart (if crash)
- 📊 CPU / RAM stats
- 🗑 File delete system
- ▶️ Multi file run
- 🌐 24/7 Hosting (Render)

📦 Requirements

- Python 3.10+
- pyTelegramBotAPI
- Flask
- psutil

⚙️ Setup

1. Install requirements

pip install -r requirements.txt

2. Set Environment Variable

TOKEN=your_bot_token

3. Run

python main.py

🌐 Deploy (Render)

- Connect GitHub repo
- Add Environment Variable (TOKEN)
- Deploy Web Service
