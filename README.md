# Telegram Hosting Bot 24/7

## Features
- Upload Python files
- Run multiple files
- Auto restart
- Delete files
- 24/7 hosting (Render)

## Commands
/start - Start bot  
run filename.py - Run file  
stop filename.py - Stop  
files - Show files  
del filename.py - Delete  

## Deploy
1. Upload to GitHub  
2. Connect to Render  
3. Add ENV:
   TOKEN=your_token
   OWNER_ID=your_id
